The Holberton School Curriculum

I am currently enrolled in the ALX Software Engineering programme, which is known for its intense, project-based curriculum. Unlike traditional schooling, we focus on peer learning and "doing hard things." Every day presents a new challenge, whether it is managing memory in C programming or understanding complex algorithms. This environment has taught me resilience and the importance of collaboration.

The program covers a wide range of topics including low-level programming, high-level languages like Python, DevOps, and System Engineering. We are trained to look under the hood of software, understanding not just how to write code, but why it works the way it does. This deep dive into computer science fundamentals is preparing me for a long-term career in the ever-evolving tech industry.

Recent Projects

  • Simple Shell: A custom command-line interpreter built from scratch using C.
  • AirBnB Clone: A full-stack web application cloning the core features of AirBnB.
  • Binary Trees: An implementation of binary trees and various traversal algorithms.

These projects have pushed me to think critically and debug systematically. For instance, working on the Simple Shell project required a solid grasp of process creation and memory management. It was a difficult journey, but seeing the shell execute commands successfully was an incredibly rewarding moment that solidified my love for backend development.
